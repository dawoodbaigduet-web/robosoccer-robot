# RoboSoccer Robot

A RoboSoccer robot developed for university robotics competitions using Arduino Nano and embedded systems.

## Features
- Wireless robot control
- Fast movement and navigation
- Compact robot design
- Motor control using Arduino Nano
- Competition-ready design

## Components
- Arduino Nano
- Motor Driver
- DC Motors
- Wheels
- Battery
- Chassis

## Skills Used
- Arduino Programming
- C++
- Embedded Systems
- Electronics
- Motor Control
- Circuit Design
- Robotics
- Teamwork

## Competitions
This robot was developed and will used in university robotics competitions
